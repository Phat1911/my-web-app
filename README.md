#  Responsibility Tracker Web App

## Overview
This is my first serious full-stack project, built to practice real-world backend and frontend development.

The application allows users to take responsibility for completing daily tasks before they expire. Each day, a new task is generated. When a user completes a task, they earn points, and lucky users may appear on the top score leaderboard.

The project focuses on authentication, file handling, database integration, and a modern frontend UI.


## Features
- User authentication
- Daily responsibility/task system
- Task expiration handling
- User score tracking
- Top score leaderboard
- Avatar upload and processing


## Tech Stack

### Backend
- Node.js
- Express
- Prisma ORM
- Multer (file upload)
- File System (`fs`)
- JWT Authentication

### Frontend
- Next.js
- Tailwind CSS


## Getting Started (Development)

### Prerequisites
- Node.js (v18+ recommended)
- npm

### Installation
1. Clone the repository:
   ```bash
   git clone <your-repo-url>
   ```

2. Install dependencies for backend:
   ```bash
   cd back-end
   npm install
   npm run dev
   ```

3. Install dependencies for frontend:
   ```bash
   cd ../front-end
   npm install
   npm run dev
   ```
